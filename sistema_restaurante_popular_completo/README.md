# Sistema Restaurante Popular

Protótipo web do Sistema de Informação Gerencial do Restaurante Popular.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Login de teste

- admin / 1234
- gestor / 1234
- estoque / 1234
- doacoes / 1234
- producao / 1234

## Atualização feita

Ao registrar uma nova produção de refeições, o sistema reduz automaticamente o estoque com base em uma ficha técnica média por refeição.
